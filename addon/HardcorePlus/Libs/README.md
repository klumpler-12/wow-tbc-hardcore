# Required Libraries

Download these from their official sources and place each in its own subfolder:

## Ace3 Suite (https://www.wowace.com/projects/ace3)
- `LibStub/LibStub.lua` — https://www.wowace.com/projects/libstub
- `CallbackHandler-1.0/CallbackHandler-1.0.lua`
- `AceAddon-3.0/AceAddon-3.0.lua`
- `AceDB-3.0/AceDB-3.0.lua`
- `AceEvent-3.0/AceEvent-3.0.lua`
- `AceTimer-3.0/AceTimer-3.0.lua`
- `AceComm-3.0/AceComm-3.0.lua`
- `AceSerializer-3.0/AceSerializer-3.0.lua`
- `AceGUI-3.0/AceGUI-3.0.lua`
- `AceConfig-3.0/AceConfig-3.0.lua` (includes AceConfigDialog, AceConfigRegistry, AceConfigCmd)
- `AceConsole-3.0/AceConsole-3.0.lua`

## Data Broker
- `LibDataBroker-1.1/LibDataBroker-1.1.lua` — https://www.wowace.com/projects/libdatabroker-1-1
- `LibDBIcon-1.0/LibDBIcon-1.0.lua` — https://www.wowace.com/projects/libdbicon-1-0
