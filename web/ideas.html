MechanicBrainstorm

PvP Death Penalties

PvP deaths should have unique penalties — not just ban or delete. Make PvP engaging within HC rules.

  • Score deductions scaled by opponent's level
  • Temporary gear lock (can't upgrade for X minutes)
  • Bounty system — killing high-score players earns bounty
  • Separate PvP ELO leaderboard
MechanicConcept

Score-Based Item Unlocks

Reaching scoring milestones unlocks access to higher gear tiers.

  • "Must earn X points before equipping Epic items"
  • Addon monitors equip events and enforces restrictions
  • Creates natural gating without arbitrary locks
MechanicConcept

Gear Battle Duels

Global opt-in mechanic (not mode-specific). Challenge another player, each wagers an item. Winner earns score + title.

  • Custom UI for selecting wagered item
  • Both players confirm before duel starts
  • Loser's item auto-deleted by addon
  • Exclusive titles for win streaks
Anti-CheatResearched

Backend Verification Layer

Blizzard's Item Restoration lets players recover deleted items every 24h — a cheating vector. Companion app solves this.

  • Companion app reads SavedVariables, posts to backend
  • Inventory hash snapshots detect restored items
  • Two-step verification: app must run before WoW launch
  • Optional — only for tournaments needing integrity

BusinessResearched

Monetization Concepts

Sustainable revenue without paywalling core HC functionality. Based on successful WoW private server models.

  • Cosmetic Cash Shop: Sell custom mounts, pets, and transmogs (most popular model, highly profitable, €200k+/yr for mid-sized servers).
  • Twitch Extension Revenue: Bits integration where viewers explicitly pay to trigger specific minor in-game events or visual effects via the overlay.
  • Unlock fee for custom private modes (not HC Classic/Plus)
  • Premium streamer tools: overlays, custom viewer controls
  • Risk: Selling direct player advantages (bags, boosts) is lucrative but controversial and risks Blizzard legal action.

Anti-Boost Detection

Optional toggle to prevent power-leveling. Detects when a high-level player is carrying.

  • Grouped with player 20+ levels higher who uses abilities near you
  • Quest auto-abandoned if completed under boosted conditions
  • Configurable threshold (guild can set level gap)
  • Togglable per mode — strict for HC Classic, optional for HC Plus

Achievement Mode

Solo-focused HC variant. Track specific milestones and challenges — for adventurers who want personal goals.

  • Pre-defined lists: "Clear all TBC dungeons solo", "Reach 70 no deaths"
  • Custom checklists players can define themselves
  • Progress tracking with timestamps and proof logs
  • Shareable achievement cards for profiles and streams
MechanicBrainstorm

Randomized Party Hardcore

Spin-the-wheel chaos mode. Random events, constraints, and viewer triggers — never the same run twice.

  • Zone Lock: Random zone + random objective — can't leave until done
  • Timer Events: New constraint every X minutes (melee only, no healing)
  • Viewer Triggers: Chat spins the wheel for next event
  • Stacking Modifiers: Events stack — "melee only + zone locked + no potions"

Guild Masters or TBCHC Admins can revoke a death if caused by griefing. Targeted at OF3 and moderated events.

  • GM or TBCHC Admin reviews death and votes to revoke

  • Primarily designed for OF3 tournament play

⚖️ Legal & TOS Considerations

  • Blizzard Addon Policy: Addons cannot be placed behind a paywall. The base HC tracking addon must be 100% free. Monetization must come through external services (Twitch Extensions, website premium features).
  • Twitch API Integration: Triggering in-game events via Twitch Bits is technically viable but falls into a gray area if it directly alters the WoW game client state. We circumvent this by having the addon read a local file or pull from an API, acting as an overlay prompt rather than an automated macro execution.
  • Server Hosting: If hosted as a Private Server, it blatantly violates Blizzard TOS. The addon-only approach ensures players use official servers, mitigating legal takedown risks.

📈 Strategic Opportunities

  • Nullcore Scoring: No existing addon dynamically tracks speed-running, under-gearing, and "first clear" bonuses in TBC. This is our massive blue-ocean feature.
  • Super Guilds: Large streamer communities want to compete. By offering built-in "Guild vs Guild" modules, we capture the esports/tournament market that manual tracking currently struggles to support.
