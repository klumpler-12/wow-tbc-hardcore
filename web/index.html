Houses

Your Guild, Many Banners

Split your guild into competing houses — sub-guild fractions with automated scoring tracking. Each house holds up to ⅛ of your guild's member cap. Need more room? Link as many guilds as you want as a Super Guild to manage them as one through the addon.

🔄 Second Chance

The Soft Reset

Concept: For players with limited playtime who hit permadeath. Instead of full character deletion, allow a “soft reset” if the character reached level 60+ before dying.

💀Permadeath
→
🗑Strip All Gear, Professions & Gold
→
🔄Continue as “Non-HC”
📄 Character permanently marked as “Permadeath — Soft Reset” on profile and guild roster. Visible to all players. No hiding it.

Rules: Must have reached 60 before death. Guild can enable/disable this option. All gear, gold, professions and earned items removed. Score reset to 0. Character flag is permanent. Needs further design on limitations.

🔌 Architecture

Modular Plugin System

Concept: The addon is split into a lightweight core + on-demand plugins. Each plugin is activated through the main addon menu only when needed. This keeps memory footprint minimal and allows features to be independently updated, disabled, or swapped for new expansion ports (WotLK, Classic+).
