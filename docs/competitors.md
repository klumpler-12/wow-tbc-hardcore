# Competitor Analysis

---

## Sauercrowd (German HC Event Addon)

**Strengths:** Solid death tracking, chat prefix system, streamer features, PvP flagging warnings, active community.

**Weaknesses:** Hardcoded rules (no GM customization), no scoring, no houses, Classic Era only, no web backend, German-only, no creative punishments.

---

## HardcoreTBC (by mocktailtv, 33k+ downloads)

**Strengths:** TBC-specific, established user base, death = deletion enforcement, lightweight.

**Weaknesses:** No multi-mode, no scoring, no rule customization, no guild features beyond death notifications, no houses, no web backend, no punishments/rewards.

---

## Base HC Addon (Classic Era)

**Strengths:** Massive community, well-maintained, comprehensive death verification, classichc.net integration, mature codebase.

**Weaknesses:** Classic Era only (no TBC plans), rigid rules, no scoring beyond alive/dead, no houses, no creative punishments.

---

## Deathlog

**Strengths:** Excellent death UI and statistics, faction-wide notifications, historical heatmaps, clean interface.

**Weaknesses:** Tracking only — no enforcement, no penalties, no scoring, no guild features, no customization.
