# Reward System

Rewards incentivize positive behavior and create memorable moments.

---

## Reward Types

### Public Rewards
Announced to guild when earned. Displayed on profile and leaderboard.
Examples: "First to kill Gruul," "Survived leveling without a single death," "Top DPS in Prince kill."

### Hidden Rewards
Secret criteria — not announced until earned. Create surprise moments.
Examples: "Die to a critter," "Fish for 2 hours straight," "Visit every flight path in Outland in one session."

### GM-Created Rewards
Defined via web dashboard. Triggered manually or on specific conditions.
Can be tied to boss kills, dungeon clears, achievements, or arbitrary triggers.

### Guild-Wide Achievements
Announced to entire guild with special notification.
Examples: "All fraction members reached level 70," "Guild cleared Karazhan with zero deaths."

### Point-Redeemable Rewards
Points accumulated through gameplay redeemable for guild rewards.
Example: Winning fraction gets first BiS gear pick from guild bank.
GM configures reward catalog and point costs.
