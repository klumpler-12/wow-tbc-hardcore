# Web Interface (Premium)

Dashboard provides guild management tools beyond what's possible in the WoW addon UI.

---

## Dashboard Sections

### Guild Overview
- Total/active members, fractions, season standings
- Recent activity feed (deaths, achievements, punishments)
- GM quick actions (trigger punishment, announce reward, start mini-game)

### Ruleset Manager
- Visual editor with toggles, sliders, dropdowns
- Comparison tool (diff two rulesets side by side)
- Export/import with shareable links
- Version history with rollback

### Leaderboard & Statistics
- Player leaderboard with filters (fraction, class, level bracket)
- Fraction leaderboard with historical charts
- Death statistics (zone heatmaps, cause breakdown, deadliest mobs)
- Progression timeline (guild progress through raid tiers)
- Player profiles with achievement showcases

### Punishment & Reward Designer
- Custom punishment templates (descriptions, durations, verification)
- Custom reward definitions with trigger conditions
- Schedule automated punishments/rewards
- View punishment/reward history

### Fraction Management
- Create, edit, delete fractions
- Assign leaders and officers
- Draft interface with drag-and-drop
- Budget tracker for draft system

### Voting System
- Heroic difficulty rating votes with live results
- Scoring system vote
- Custom polls
- Vote history and analytics

### Analytics (Deep Dive)
- Player performance trends over time
- Optimal group compositions from historical data
- Risk assessment (death probability by class/spec)
- Engagement metrics (session frequency, playtime trends)
